2. 提出适合高维空间单位球均匀采样的算法（用伪代码形式描述）。

---


伪代码：在每个维度使用高斯采样，然后整体向量归一，再乘以一个适当调整的半径，由于高斯分布的球对称性，所以可以实现单位球内均匀采样。
  ```
  For i = 1 to N:
    Let z be a d-dimensional vector where each component z_j ~ N(0, 1)// 从标准正态分布中生成一个 d 维向量 z

    norm_z = sqrt(sum(z_j^2 for j=1 to d))
    u = z / norm_z//归一化

    Let U ~ Uniform(0, 1)
    
    r = U^(1/d)//确保选择的半径符能够在高维空间中均匀分布
    
    x = r * u
    Append x to Samples
```
