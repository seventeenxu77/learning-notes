##### 1. 证明：解密一个Feistel密码相当于密文使用加密算法，但密钥编排方案要逆序使用。

**加密操作（第 i 轮）**  
输入：$(L_{i-1}, R_{i-1})$，密钥：$K_i$
    $L_{i} = R_{i-1}$
    $R_{i} = L_{i-1} \oplus F(R_{i-1}, K_i)$
已知信息： 
$R_{i-1} = L_{i}$

利用 **异或自反性**（即 $(A \oplus B) \oplus B = A$），对等式两边同时异或 $F(R_{i-1}, K_i)$：  
    $L_{i-1} = R_{i} \oplus F(R_{i-1}, K_i)$

将步骤A中得到的 $R_{i-1} = L_{i}$ 代入上式：  
    $L_{i-1} = R_{i} \oplus F(L_{i}, K_i)$


对比加密与解密的核心公式：
- 加密：$L_i=R_{i-1}$，$R_i=L_{i-1}\oplus F(R_{i-1},K_i)$  
- 解密：$R_{i-1}=L_i$，$L_{i-1}=R_i\oplus F(L_i,K_i)$  

可见，解密时只需将密文 $(L_i,R_i)$ 作为输入，沿用加密轮函数 $F$ 和当前密钥 $K_i$，即可还原前一轮的明文 $(L_{i-1},R_{i-1})$。若将多轮加密的密钥序列记为 $K_1,K_2,...,K_n$，则解密需逆序使用密钥 $K_n,K_{n-1},...,K_1$，即“密文用加密算法，密钥逆序”。

##### 2. 设DES(x,K)表示使用密钥K对消息x进行DES加密。假定记y= DES(x,K), y’= DES(c(x),c(K))，这里c(∙)表示对其自变量按比特取补。试证明：y’=c(y), 即如果明文和密钥都按比特取补，则密文同样按比特取补。



**定理：** 设 $y = \text{DES}(x, K)$，则 $\text{DES}(c(x), c(K)) = c(y)$，其中 $c(\cdot)$ 表示比特取补。

##### 首先证明
$$c(A) \oplus c(B) = A \oplus B$$
对于A和B的每一位，考虑以下两种情况：
- 若 $A_i = B_i$，则 $c(A_i) = c(B_i)$，所以 $c(A_i) \oplus c(B_i) = 0 = A_i \oplus B_i$。
- 若 $A_i \neq B_i$，则 $c(A_i) \neq c(B_i)$，所以 $c(A_i) \oplus c(B_i) = 1 = A_i \oplus B_i$。
证毕。

##### 16 轮 Feistel 迭代（归纳证明）

假设在 $i-1$ 轮成立 $L'_{i-1} = c(L_{i-1})$ 且 $R'_{i-1} = c(R_{i-1})$。

**证明 $L'_i = c(L_i)$：**
    $$L'_i = R'_{i-1} = c(R_{i-1}) = c(L_i) \quad \text{（成立）}$$

**证明 $R'_i = c(R_i)$：**
    $$R'_i = L'_{i-1} \oplus F(R'_{i-1}, K'_i) \quad $$
    $$R'_i = c(L_{i-1}) \oplus F(c(R_{i-1}), c(K_i)) \quad $$
    $$R'_i = c(L_{i-1}) \oplus c(F(R_{i-1}, K_i)) \quad $$
    $$R'_i = c(L_{i-1} \oplus F(R_{i-1}, K_i)) \quad $$
    $$R'_i = c(R_i) \quad \text{（成立）}$$



##### 经过 16 轮迭代和最终逆置换 $\text{IP}^{-1}$：

$$y' = c(\text{IP}^{-1}(R_{16}, L_{16}))$$
由于 $\text{IP}^{-1}(R_{16}, L_{16}) = y$，故：
